    ConvertView

    Extends:

    AbstractHierarchy → ConvertView

    漢字変換ビューを表すクラス。 それぞれ一つの文節を担当し、複数の漢字変換候補を持ちます。 また、内部には変換候補としてRowクラスのインスタンスを持ちます
